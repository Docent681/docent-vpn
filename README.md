# Docent-Vpn - Веб-интерфейс для управления своим Vpn-сервером
Данный проект представляет собой веб интерфейс, позволяющий автоматизировать и облегчить использование и администрирование своего Vpn сервера на базе Outline Vpn от Jigsaw.

## Требования
- Сервер на одном из дистрибутивов Linux (проверено на Ubuntu Server) 
- Git (опционально, если устанавливаете архивом)

## Указания к установке
Установите архив с проектом или войдите в домашний каталог и выполните следующее:
``` 
git clone https://github.com/Docent681/docent-vpn
```
после этого запустите следующий скрипт с правами суперпользователя:

``` 
sudo ~/docent-vpn/setup.sh
```
После этого будет установлен Outline Vpn, по желанию можете автоматически настроить фаерволл.
